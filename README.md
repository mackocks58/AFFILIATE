# Eagle Star Affiliate Platform

Welcome to the **Eagle Star Affiliate Platform**, a premium, high-performance web application designed for betting enthusiasts and affiliates. This platform combines a sleek React frontend with a robust Express backend and Firebase integration to deliver a seamless user experience.

## 🚀 Key Features

- **Dynamic Betting Interface**: A modern, responsive UI built with React and Vite.
- **Affiliate Management**: Comprehensive tools for managing affiliate relationships and tracking performance.
- **Real-time Data**: Integrated with Firebase for live updates and secure data management.
- **Secure Authentication**: Built-in user authentication and role-based access control.
- **Premium Aesthetics**: Designed with a focus on visual excellence and user engagement.

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite, TypeScript
- **Backend**: Node.js, Express
- **Database/Hosting**: Firebase
- **Styling**: Vanilla CSS (Custom UI Design)

## 📦 Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mackocks58/AFFILIATE.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env` file based on `.env.example` and add your configuration.

4. **Run the development server**:
   ```bash
   npm run dev
   ```

## 📜 Scripts

- `npm run dev`: Starts both the Vite frontend and Express server concurrently.
- `npm run build`: Builds the production-ready frontend.
- `npm start`: Runs the Express server.
- `npm run deploy:hosting`: Builds and deploys to Firebase Hosting.
